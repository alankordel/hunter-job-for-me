Projeto desenvolvido para coletar, transformar e armazenar vagas de emprego compativeis com meu automaticamente, utilizando técnicas de Web Scraping e Engenharia de Dados.

## Objetivos

- Coletar vagas de diferentes plataformas
- Filtrar vagas por perfil profissional
- Armazenar em banco de dados
- Disponibilizar dashboard
- Automatizar o processo de atualização
Será adicionado mais objetivos conforme for identificado
